Set up

Before we get started we first need to set up our MIDI (Musical Instrument Digital Interface) controller with our laptop and the Trakor software. To the right shows how the MIDI controller uses the software. It does not actually handle any of the audio on board but is all done through the software itself. The MIDI controller basically controls the software.

I/O (Input/Output)

The outputs on the MIDI controller can output audio to various devices such as amps, speakers or headphones. The user has a headphone jack for monitoring the audio which can be turned on and off using this button.



Along with the outputs and monitors you can record the audio directly through the software. This utilises your laptops sound card and records exactly what Traktor outputs. All of these outputs can be changed within Traktor to suit your needs.
