Cue Points (Hotcue)

Now that you have two tracks playing you want them to be in time with each other. This is easily achieved by using the sync hover01

The sync button will beat match the two tracks and allow you to seamlessly transition from one to another. You can also select which is the master tempo by hitting the hover02.

As well as the automatic sync button you can manually adjust the tempo on each deck. You can choose if you want to lock the pitch with this hover03.

Within Traktor you can set cue points on your tracks to make is easier to get to the section of the song that you want. There are 2 different kinds of cue points. You have the reference cue points and hot cues.

The first kind of reference cue point is the beat grid (white). When you load a track into Traktor it automatically creates a beat grid for you, but if it is out you can manually adjust it yourself. If your beat grid is out then your mixing will be too.

The next one is the LOAD (yellow) marker. This marker sets where the track will load to when it is loaded onto a deck. You can have multiple LOAD markers but only one will work.

The third cue type is the fade out (orange) cue point. This cue point is purely for reference and does not actually do anything. You can have as many fade out cue points as you like to show the possible fade out points for the song.

The next cue type if the fade in (orange) cue point. This cue point is like the fade out but instead shows any possible fade it points. You can add multiple fade in points to show at which points of the song you can fade into.

The final reference cue point is just a standard cue point (blue). These cue points are a standard cue point that can be used freely to mark any part of the song. These are usually used to mark certain phrases of a song so you know what is happening whilst the track is playing.

Hotcue Points

The hotcue cue points are mainly used for two different ways. Hotcue juggling and marking phrases.

Hotcue juggling involves setting hotcues to certain notes of a song to add the ability to remix tracks on the fly.

Using the hotcues of mark certain phrases of the song allows you to quickly jump from one part of the song to another. This can work alongside the reference cues.
